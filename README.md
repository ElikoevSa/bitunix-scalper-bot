# Bitunix Scalper Bot

Торговый робот-скальпер для биржи Bitunix с веб-панелью управления.

## Особенности

- **7 торговых стратегий скальпинга:**
  1. RSI Scalping - торговля на основе индикатора RSI
  2. Bollinger Bounce - торговля на отскоках от полос Боллинджера
  3. EMA Crossover - торговля на пересечениях экспоненциальных скользящих средних
  4. Momentum Breakout - торговля на пробоях с высоким объемом
  5. Support/Resistance - торговля на уровнях поддержки и сопротивления
  6. Volume Spike - торговля на всплесках объема
  7. Mean Reversion - торговля на возврате к среднему

- **Автоматический технический анализ** для выбора торговых пар
- **Веб-панель управления** с отображением:
  - Выбранной стратегии
  - Выбранной торговой пары
  - Цены входа и выхода
  - Профита/убытка с учетом комиссий
  - Статистики торговли
- **Интеграция с Bybit Demo Trading** для безопасного тестирования стратегий

## Технические требования

- Java 11+
- Maven 3.6+
- Spring Boot 2.7.18

## Установка и запуск

1. **Клонируйте проект:**
   ```bash
   git clone <repository-url>
   cd bitunix-scalper-bot
   ```

2. **Настройте API ключи:**
   
   **Для демо-торговли (рекомендуется для начала):**
   - Войдите в Bybit и переключитесь в режим "Demo Trading"
   - Создайте API ключ для демо-торговли
   - Настройте в `application.yml`:
   ```yaml
   bitunix:
     api:
       base-url: https://api-demo.bybit.com
       api-key: YOUR_DEMO_API_KEY
       secret-key: YOUR_DEMO_SECRET_KEY
   ```
   
   **Для реальной торговли:**
   Создайте файл `application-local.yml` в папке `src/main/resources/`:
   ```yaml
   bitunix:
     api:
       api-key: YOUR_API_KEY
       secret-key: YOUR_SECRET_KEY
   ```
   
   Подробнее о демо-торговле см. [DEMO_TRADING.md](DEMO_TRADING.md)

3. **Соберите проект:**
   ```bash
   mvn clean install
   ```

4. **Запустите приложение:**
   ```bash
   mvn spring-boot:run
   ```

5. **Откройте веб-панель:**
   - Основная панель: http://localhost:8080
   - Демо-торговля: http://localhost:8080/demo/dashboard

## Конфигурация

### Комиссии биржи
- Maker: 0.0200% (0.0002)
- Taker: 0.0600% (0.0006)

### Настройки торговли
- Максимальный размер позиции: 1000 USDT
- Стоп-лосс: 0.5%
- Тейк-профит: 0.3%
- Максимальный дневной убыток: 100 USDT

## Торговые стратегии

### 1. RSI Scalping
- **Вход:** RSI < 30 (перепроданность) или RSI > 70 (перекупленность)
- **Выход:** RSI возвращается к 50
- **Приоритет:** 5

### 2. Bollinger Bounce
- **Вход:** Цена касается нижней/верхней полосы
- **Выход:** Цена возвращается к средней линии
- **Приоритет:** 4

### 3. EMA Crossover
- **Вход:** Золотой крест (EMA12 > EMA26) или Медвежий крест (EMA12 < EMA26)
- **Выход:** Обратное пересечение
- **Приоритет:** 3

### 4. Momentum Breakout
- **Вход:** Объем > 150% от среднего + ценовой импульс
- **Выход:** Объем падает ниже 80% от среднего
- **Приоритет:** 6

### 5. Support/Resistance
- **Вход:** Цена касается уровня поддержки/сопротивления
- **Выход:** Цена достигает противоположного уровня
- **Приоритет:** 2

### 6. Volume Spike
- **Вход:** Объем > 200% от среднего + движение цены
- **Выход:** Объем падает ниже 50% от среднего
- **Приоритет:** 7

### 7. Mean Reversion
- **Вход:** Цена отклоняется на 2+ стандартных отклонения от среднего
- **Выход:** Цена возвращается к среднему
- **Приоритет:** 1

## Веб-панель

Веб-панель предоставляет:

1. **Панель торговли:**
   - Текущая стратегия
   - Выбранная торговая пара
   - Цена входа/выхода
   - Профит/убыток
   - Кнопки управления

2. **Статистика:**
   - Общий профит
   - Процент успешных сделок
   - Общее количество сделок
   - Количество успешных сделок

3. **Таблица торговых пар:**
   - Список доступных пар
   - Цены и объемы
   - Технические индикаторы
   - Статус активности

4. **Стратегии:**
   - Список всех стратегий
   - Статус активности
   - Приоритеты

- Управляет рисками
- Ведет журнал всех сделок
