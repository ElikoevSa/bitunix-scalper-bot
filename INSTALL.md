# Инструкция по установке Bitunix Scalper Bot

## Требования

1. **Java 11 или выше**
   - Скачайте с https://adoptium.net/
   - Установите и добавьте в PATH

2. **Apache Maven**
   - Скачайте с https://maven.apache.org/download.cgi
   - Распакуйте и добавьте в PATH

3. **API ключи Bitunix**
   - Зарегистрируйтесь на https://bitunix.com
   - Создайте API ключи в личном кабинете

## Установка

### Вариант 1: Автоматический запуск (Windows)

1. Откройте командную строку в папке проекта
2. Запустите `start.bat`
3. Откройте браузер: http://localhost:8080

### Вариант 2: Ручной запуск

1. **Настройте API ключи:**
   Создайте файл `src/main/resources/application-local.yml`:
   ```yaml
   bitunix:
     api:
       api-key: YOUR_API_KEY_HERE
       secret-key: YOUR_SECRET_KEY_HERE
   ```

2. **Скомпилируйте проект:**
   ```bash
   mvn clean compile
   ```

3. **Запустите приложение:**
   ```bash
   mvn spring-boot:run
   ```

4. **Откройте веб-интерфейс:**
   http://localhost:8080

## Настройка

### API ключи
1. Войдите в личный кабинет Bitunix
2. Перейдите в раздел API
3. Создайте новый API ключ с правами:
   - Чтение данных
   - Торговля (только для реальной торговли)
4. Скопируйте API ключ и секретный ключ
5. Вставьте их в файл `application-local.yml`

### Настройки торговли
Отредактируйте файл `src/main/resources/application.yml`:

```yaml
trading:
  risk:
    max-position-size: 1000.0      # Максимальный размер позиции
    stop-loss-percentage: 0.5      # Стоп-лосс в %
    take-profit-percentage: 0.3    # Тейк-профит в %
    max-daily-loss: 100.0          # Максимальный дневной убыток
```

## Использование

1. **Запустите бота** через веб-интерфейс
2. **Выберите стратегию** из списка доступных
3. **Нажмите "Start Trading"** для начала торговли
4. **Мониторьте результаты** в реальном времени

## Безопасность

⚠️ **ВАЖНЫЕ ПРЕДУПРЕЖДЕНИЯ:**

- Используйте только тестовые API ключи для начала
- Ограничьте размер позиций
- Настройте стоп-лоссы
- Регулярно проверяйте работу бота
- Не оставляйте бота без присмотра

## Устранение неполадок

### Ошибка "Java not found"
- Установите Java 11+
- Добавьте Java в переменную PATH

### Ошибка "Maven not found"
- Установите Apache Maven
- Добавьте Maven в переменную PATH

### Ошибка подключения к API
- Проверьте правильность API ключей
- Убедитесь в наличии интернет-соединения
- Проверьте статус API Bitunix

### Ошибка компиляции
- Убедитесь, что Java 11+ установлена
- Проверьте версию Maven (3.6+)
- Очистите проект: `mvn clean`

## Поддержка

При возникновении проблем:
1. Проверьте логи в консоли
2. Убедитесь в правильности настроек
3. Проверьте подключение к интернету
4. Обратитесь к документации Bitunix API
